# BoaDica SSD Dashboard 🚀

Este é um Web App inteligente para monitoramento e análise de preços de SSDs no mercado do Rio de Janeiro (BoaDica).

## 🌟 Funcionalidades
- **Automação Total:** Dados atualizados automaticamente via GitHub Actions (3x ao dia).
- **Cálculo de ROI:** Sistema baseado em Markup Divisor com escala de cores.
- **Filtros Avançados:** Busca por região, tipo (BOX/OEM) e ordenação por popularidade.
- **Design Moderno:** Interface responsiva com Glassmorphism.

## 🤖 Como funciona a atualização?
O robô (`scraper.js`) roda nos servidores do GitHub seguindo este cronograma:
- 06:00 AM (UTC)
- 12:00 PM (UTC)
- 18:00 PM (UTC)

## 🛠️ Tecnologias
- HTML / CSS / JavaScript
- Node.js (Scraper)
- GitHub Actions (Automatização)
- Vercel (Hospedagem)
